# Involvex User Guide

Welcome to the official user guide for **Involvex**, the hacker-style dashboard for tracking what's trending in the open-source world.

Involvex helps you stay ahead of the curve by monitoring GitHub repositories and npm packages that are gaining momentum.

## What is Involvex?

Involvex is a multi-platform tool (Mobile & Web) designed for developers who want to discover new tools, libraries, and frameworks as they become popular. Whether you're looking for the next big React framework or a handy CLI utility, Involvex has you covered.

### Key Highlights
- **Real-time Trending:** Fetch daily, weekly, and monthly trending items.
- **Unified Interface:** Track both GitHub and npm in one place.
- **Subscriptions:** "Watch" projects and get notified about new releases.
- **Hacker Aesthetic:** A clean, Matrix-inspired dark green terminal theme.
