# Usage Guide

Getting started with Involvex is easy. Here's how to make the most of the app.

## Searching for Projects
Use the **Search** tab to find specific repositories or packages.
- Type the name or a keyword (e.g., "react", "tailwind").
- Switch between GitHub and npm search results using the toggle.

## Viewing Trends
The **Home** screen shows you what's currently popular.
- Tap on any item to see its description, stars, and other metadata.
- Change the timeframe (Daily/Weekly/Monthly) in the **Settings** tab.

## Subscribing to Updates
When you find a project you like, tap the **Subscribe** (plus icon) button.
- It will now appear in your **Subscriptions** tab.
- Involvex will periodically check for new releases and update your feed.

## Customizing the Theme
Go to the **Settings** tab to tweak the appearance.
- Toggle between the "Hacker Theme" and a standard Dark/Light mode.
- Adjust notification preferences and data refresh intervals.
