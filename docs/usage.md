# > Terminal Operations_

Getting started with Involvex is efficient. Here's how to navigate the system.

## > Searching Protocols
Use the **Search** module to find specific repositories or packages.
- Input name or keyword (e.g., "react", "tailwind").
- Switch between GitHub and npm search results.

## > Viewing Data Streams
The **Home** screen displays current market trends.
- Tap on any node to see its metadata.
- Adjust timeframe parameters in the **Settings** tab.

## > Subscribing to Updates
When you identify a project, tap the **Star** icon.
- It will be added to your **Subscriptions** feed.
- Involvex will periodically monitor for new release tags.

## > Modifying UI Protocols
Go to the **Settings** tab to tweak appearance.
- Toggle between "Hacker Theme" and standard modes.
- Adjust notification priority and data refresh cycles.
